# Library Management System

Проект системы управления библиотекой на Java.

## Описание
Консольное приложение для управления книгами в библиотеке.

## Структура проекта
- `src/main/java/library/Book.java` - класс книги
- `src/main/java/library/Library.java` - основной класс библиотеки
- `src/main/java/library/Main.java` - точка входа в приложение

## Требования
- JDK 17+
- Git 2.30+

## Установка
```bash
git clone <repository-url>
cd library-management-system
javac src/main/java/library/*.java
java -cp src/main/java library.Main
